Validates the value of an option is allowed for that option

This inspection validates that the value supplied matches what is allowed for the option.

Only a few types are actually validated presently:

  • Options which take a boolean argument.
  • Options which take a mode argument (e.g., permission mask).
  • The Service/Type= option.
  • The Service/Restart= option.
  • The Service|Socket|Mount|Swap/KillMode= option.
  • The Unit/Description= option.
